ℹ️ How to Use:

  • Step 1: Click "Download Macro File" to download the Excel file
  • Step 2: Click "Run Macro" to open the downloaded file
  • The file will be automatically unblocked when you click "Run Macro"
  • The file is saved in your Downloads folder

⚠️ Important Notes:

  • After opening, click "Enable Content" or "Enable Macros" in Excel
  • Make sure Excel is installed on your computer
  • The macro requires permissions to access Azure DevOps data

🔒 If Macros Are Still Blocked (Manual Fix):

  1. Go to your Downloads folder: C:\Users\{YourName}\Downloads
  2. Right-click on Regression_Data.xlsm
  3. Select "Properties" from the menu
  4. At the bottom, check the box that says "Unblock"
  5. Click "Apply" then "OK"
  6. Now open the file - macros should work!
