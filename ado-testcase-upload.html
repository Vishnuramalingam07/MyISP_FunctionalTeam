About ADO Testcase Upload

📁 Excel Format: Sheet named "Test Cases"
🏢 Organization: accenturecio08
📌 Project: AutomationProcess_29697
📎 File Types: .xlsx, .xlsm, .xls
  • Reads test cases from the Excel workbook and uploads them to the specified ADO Test Suite
  • Links each test case to its User Story (from the User_StoryID column)
  • Assigns test cases to the configured Lead and sets all required fields
  • Generates a detailed upload summary report after completion
