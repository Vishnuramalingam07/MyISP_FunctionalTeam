🔗 Auto External Reference ID Update

Batch update ExternalRef IDs on ADO work items using M POC Mapping data

What this does

  • 📋 Reads the ADO Query to fetch all target work items
  • 🗺️ Loads the M POC Mapping CSV (Node Name → ExternalRef ID)
  • 🔍 Matches each work item's Area Path to the mapping
  • ⚡ Batch-fetches work item details in parallel (100 per request)
  • ✏️ Updates Custom.ExternalRefID field on matched items
  • ⏭️ Skips items already set to the correct value
  • 💾 Saves a CSV result file to the M_POC folder
⚙️ Enter the ADO Query ID in the field below, or save it permanently via Settings → US Ext Ref ID Update.
The AZURE_DEVOPS_PAT environment variable must be set on this machine.
