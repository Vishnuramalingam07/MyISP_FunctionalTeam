🔧 HOW THE AUTO-DETECTION WORKS:
═══════════════════════════════════════════════════════════════════════

Environment Variables Used:
---------------------------
%USERNAME%      = Current logged-in user (e.g., "vishnu.ramalingam")
%USERPROFILE%   = Full path to user folder (e.g., "C:\Users\vishnu.ramalingam")
%COMPUTERNAME%  = This computer's name (e.g., "CDC2-L-G04FSGN")

Auto-Detected Path:
------------------
%USERPROFILE%\MyISP_Tools\index.html

Becomes:
-------
C:\Users\[current-user]\MyISP_Tools\index.html
