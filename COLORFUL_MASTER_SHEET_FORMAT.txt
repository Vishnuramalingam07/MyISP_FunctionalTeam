╔══════════════════════════════════════════════════════════════════════════════╗
║                        SPECIAL FEATURES                                      ║
╚══════════════════════════════════════════════════════════════════════════════╝

🔒 FREEZE PANES:
   • Frozen at cell D4 (column D, row 4)
   • This means:
     - Rows 1-3 (headers) stay visible when scrolling down
     - Columns A-C (Name, Lead, Location) stay visible when scrolling right
   • Makes it easy to navigate large datasets!

📐 COLUMN WIDTHS:
   • Column A (Team Member Names): 30 characters
   • Column B (Lead Name): 25 characters
   • Column C (Location): 22 characters
   • Date columns (1-31): 8 characters each
   • Summary columns: 12 characters each

🎯 CELL BORDERS:
   • All cells have thin, light gray borders (#cbd5e1)
   • Creates a professional grid layout
   • Makes it easy to distinguish between cells

╔══════════════════════════════════════════════════════════════════════════════╗
║                        TECHNICAL IMPLEMENTATION                              ║
╚══════════════════════════════════════════════════════════════════════════════╝

PYTHON LIBRARIES USED:
   • openpyxl.styles.PatternFill - For cell background colors
   • openpyxl.styles.Font - For text styling
   • openpyxl.styles.Alignment - For text positioning
   • openpyxl.styles.Border - For cell borders
   • openpyxl.styles.Side - For border line styles

STYLING OBJECTS CREATED:
   • header_fill, row2_fill, row3_fill - Header backgrounds
   • header_font, row2_font, row3_font - Header text styles
   • even_row_fill, odd_row_fill - Alternating row backgrounds
   • data_font - Regular text style for data cells
   • thin_border - Border definition for all cells
   • Various alignment objects for different needs

COLOR CODES EXPLAINED:
   • Hex format: #RRGGBB (Red, Green, Blue)
   • Example: #7c3aed = Purple (124 red, 58 green, 237 blue)
   • Gradient effect created with start_color and end_color

╔══════════════════════════════════════════════════════════════════════════════╗
║                        CUSTOMIZATION OPTIONS                                 ║
╚══════════════════════════════════════════════════════════════════════════════╝

Want to change the colors? Edit these values in app.py:

ROW 1 HEADER:
   header_fill = PatternFill(start_color='7c3aed', ...)  # Change '7c3aed'

ROW 2 METADATA:
   row2_fill = PatternFill(start_color='4f46e5', ...)    # Change '4f46e5'

ROW 3 DAY NAMES:
   row3_fill = PatternFill(start_color='06b6d4', ...)    # Change '06b6d4'

EVEN ROWS:
   even_row_fill = PatternFill(start_color='f0f9ff', ...)  # Change 'f0f9ff'

ODD ROWS:
   odd_row_fill = PatternFill(start_color='FFFFFF', ...)   # Keep white or change
