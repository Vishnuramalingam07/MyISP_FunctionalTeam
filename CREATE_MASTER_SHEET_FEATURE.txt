A new "Create Master Sheet" button has been added to the attendance tracker that
allows you to pre-populate a master sheet with ALL team members for any month.

╔══════════════════════════════════════════════════════════════════════════════╗
║                              HOW IT WORKS                                    ║
╚══════════════════════════════════════════════════════════════════════════════╝

1. BUTTON LOCATION
   • The "📄 Create Master Sheet" button appears in the action buttons row
   • Located between "Save to Master Excel" and "Download Excel" buttons
   • Button has a green gradient background for visibility

2. FUNCTIONALITY
   When clicked, the button will:
   
   ✓ Check if a sheet for the selected month/year already exists
   ✓ If sheet DOES NOT exist:
     - Creates a new sheet with January format (standardized)
     - Adds ALL team members from Team Details.xlsx
     - Includes Lead Name for each team member
     - Includes Location for each team member
     - Pre-formats the sheet with proper column widths
     - Adds date headers and day names
   
   ✓ If sheet ALREADY exists:
     - Shows a message: "Sheet already exists"
     - Displays current team member count
     - Suggests using "Refresh Data" to load the existing sheet

3. WHAT GETS POPULATED
   • Column A: Team Member Names (all 175+ members from Team Details)
   • Column B: Lead Name (mapped from Team Details)
   • Column C: Location (mapped from Team Details)
   • Columns D onward: Date columns (1-31 based on month)
   • Summary columns: Total Present, Total Leaves, UnPlanned, SickLeave

4. SHEET FORMAT
   • Row 1: Team Member Names | Lead Name
   • Row 2: Metadata | 📍 Location | 1 | 2 | 3 | ... | 31 | Summary columns
   • Row 3: Timestamp | Day names (Mon, Tue, Wed...)
   • Row 4+: Team member data rows (sorted by Lead, then by Name)

╔══════════════════════════════════════════════════════════════════════════════╗
║                           HOW TO USE                                         ║
╚══════════════════════════════════════════════════════════════════════════════╝

STEP 1: Open the Attendance Tracker
   • Navigate to: http://localhost:8000/Attendance/team-attendance-tracker-sharepoint.html
   • Or open from the Flask server URL

STEP 2: Select Month and Year
   • Use the month/year dropdowns to select the target month
   • Example: March 2026

STEP 3: Click "Create Master Sheet"
   • The button will show a confirmation dialog
   • Click OK to proceed

STEP 4: Wait for Confirmation
   • If sheet doesn't exist: Success message with member/lead counts
   • If sheet exists: Info message with existing sheet details

STEP 5: Load the Data (if new sheet created)
   • Select a lead from the dropdown
   • Click "Refresh Data" to load the newly created sheet

   → Verify Team Details.xlsx mapping is up to date
   → Run extract_team_data.py to refresh data
